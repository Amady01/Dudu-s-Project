# Dudu-s-Project
1. 🎯 Le Chef de Produit (PM - Product Manager)
C'est qui ? C'est le chef d'orchestre et le visionnaire de l'équipe. Si vous aimez organiser, trancher les décisions, comprendre les besoins des utilisateurs et raconter une histoire captivante, ce rôle est pour vous.
Sa mission dans le Lab : Il s'assure que le projet avance au rythme du chrono, coordonne le travail des autres et porte la voix de l'équipe lors du pitch final de 90 secondes devant le jury.
Profils idéaux : Business, Management, Marketing.
2. ✍️ Le Master Prompt Engineer
C'est qui ? C'est le "traducteur" et le psychologue de l'IA. Si vous aimez écrire de manière précise, que vous êtes pointilleux sur les mots et que vous aimez chercher la meilleure manière de formuler une idée pour obtenir exactement ce que vous voulez d'une machine, vous êtes fait pour ça.
Sa mission dans le Lab : C'est lui qui va concevoir, tester et documenter toutes les instructions (les fameux prompts Zero-Shot, Few-Shot, Markdown) dans le Journal de Prompts de l'équipe pour piloter Claude ou ChatGPT.
Profils idéaux : Communication, Marketing, Littérature, Stratégie.
3. 🛠️ Le Dev UI (No-Code)
C'est qui ? C'est le bâtisseur et l'artisan numérique. Si vous êtes curieux, que vous aimez bidouiller des outils en ligne, cliquer sur des boutons pour voir ce qui se passe et que vous voulez voir votre idée prendre vie visuellement sans écrire une seule ligne de code traditionnel, foncez.
Sa mission dans le Lab : C'est lui qui ouvre le dépôt GitHub, qui configure l'espace de travail Dify en français et qui assemblera les briques de l'application (l'interface et les agents IA) pour que le projet soit accessible en ligne via une vraie URL.
Profils idéaux : Profils curieux, orientés solutions, attirés par le design ou la technique No-Code.
4. 🌍 Le Responsable Impact (et Éthique)
C'est qui ? C'est la boussole morale et l'avocat du citoyen. Si vous vous souciez de l'impact des technologies sur la société, de la protection des données des utilisateurs locaux (comme nos maraîchers), et de la viabilité juridique ou financière d'un projet, ce rôle est capital.
Sa mission dans le Lab : Il analyse les retours des interviews d'empathie, veille à ce que l'application ne crée pas d'effets secondaires négatifs et rédige la note d'éthique IA du projet (pour s'assurer que l'IA reste responsable et utile).
Profils idéaux : Droit, Finance, RSE, Ressources Humaines.
